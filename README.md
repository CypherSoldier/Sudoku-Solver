# Sudoku-Solver

## How It Works
- '#' represents the borders
- 'O' represents the snake
- 'F' represents food
You can move the snake using 'W, A, S, D' (UP, LEFT, DOWN, RIGHT) to consume the food and grow in length.
You get 10 points for each piece of food eaten.
